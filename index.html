my discord bot

all lowercase bot with utility, moderation, and embed builder commands

invite bot

commands

utility

  • ping - check latency
  • userinfo - get info about a user
  • serverinfo - get info about the server
  • avatar - show a user's avatar

moderation

  • kick - kick a member (admin/mod only)
  • ban - ban a member (admin/mod only)
  • unban - unban a member by ID (admin/mod only)
  • mute - mute a member (admin/mod only)
  • unmute - unmute a member (admin/mod only)
  • warn - warn a member (admin/mod only)
  • warnings - check warnings (admin/mod only)
  • clear - delete messages (admin/mod only)

embed builder

  • build custom embeds with title, description, color, optional image
  • usable with both prefix and slash commands
